<b>05. while문</b></h4>
<p>반복해서 문장을 수행해야 할 경우 반복문을 사용합니다. while문은 미리 반복 횟수를 알 수 없고 조건에 따라서 반복하는 경우에 주로 사용합니다.</p>
<p>while(조건문) { </p>
<p align="center"><수행할 문장1></p>
<p align="center"><수행할 문장2></p>
<p align="center">...</p>
<p>}</p>
<p>조건문이 참인 동안 while문 아래의 문장들이 계속해서 수행됩니다.
<br>1에서 10까지의 합을 구할 때, while문을 이용한 예입니다.</p>
<img src="while.png" width=343 height=103>
<p>‘1에서 10까지의 합 : 55’ 가 출력됩니다.</p>
<p>1. 초기화 : int 변수 i는 while문 외부에서 루프가 실행되기 전에 1로 초기화됩니다. 합의 결과를 넣을 sum을 외부에서 0으로 초기화 해줍니다.
<br>2. 조건 : while 조건문은 i<=10인데 i가 10보다 작거나 같을 동안에 while문 안의 문장들을 계속 실행되고 11이 되면 while문을 빠져나오게 됩니다.
<br>3. 증감 : i++는 i값을 계속 1씩 증가시키는 역할을 합니다. i++ 때문에 1+2+3+…+10 이렇게 1씩 증가시키면서 덧셈을 할 수 있는 것입니다.</p>
