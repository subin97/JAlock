<b>03. if-else문</b></h4>
<p>if문만으로는 좀 더 복잡한 상황을 처리하는데 부족합니다. if 뒤에 조건문이 거짓일 때도 무엇인가를 실행하고 싶을 때는 if-else문을 사용합니다.</p>
<p>if(조건문) { </p>
<p align="center"><수행할 문장1></p>
<p align="center"><수행할 문장2></p>
<p align="center">...</p>
<p>} else { </p>
<p align="center"><수행할 문장A></p>
<p align="center"><수행할 문장B></p>
<p align="center">...</p>
<p>}</p>
<p>조건문이 참이면 if문 바로 다음의 문장들을 수행하고 조건문이 거짓이면 else문 다음의 문장들을 수행하게 됩니다.
<br>다음은 if-else문의 예입니다.</p>
<img src="ifnum.png" width=344 height=153>
<p>num이 0보다 큰 7이기 때문에 조건문이 참이 돼서 ‘양수입니다’가 출력됩니다. 만약 num이 –5였다면 조건문이 거짓이 돼서 ‘음수입니다’가 출력됐을 것입니다.</p>
