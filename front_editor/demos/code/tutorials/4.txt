<b>04. 데이터 형식3 : <p align="center">문자형과 문자열</b></p></h4>

<p>문자형인 char 데이터 유형은 하나의 문자를 나타내는 데 사용됩니다. 하나의 문자가 아닌 문자열을 나타낼 때는 String 타입을 사용합니다.</p>
<p>1. char을 사용할 때는 다음과 같이 작은 따옴표를 써야합니다. 예) ‘가’, ‘A’</p>
<p>2. String을 사용할 때는 다음과 같이 큰 따옴표를 써야합니다. 예) “홍길동”, “Hello world”</p>
<p>☆ 잘록에서는 따옴표를 쓰지 않고도 쓸 수 있는 print 블록을 사용하겠습니다.</p>

<p>2. print 블록에 String형 문자열 “안녕 자바”를 입력하세요.</p>
