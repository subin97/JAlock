<b>05. 변수</b></h4>
<p>변수는 값을 저장하는 기능을 하는데 사용하기 전에 반드시 미리 선언해야 합니다. 변수를 선언하는 방법은 원하는 자료형을 쓰고 이어서 변수 이름을 쓰면 됩니다.</p>
<p>1. 자료형인 int와 이름인 size를 이어서 int size; 이렇게 쓰면 됩니다. 여기서 세미콜론 ; 은 하나의 문장을 종료하는데 사용됩니다.</p>
<p>2. int size=7; 이렇게 쓰면 size라는 변수에 int형인 7을 저장하게 됩니다.</p>
<p>3. 변수의 이름은 문자와 숫자의 조합으로 만들어지고 한글도 가능합니다. 다만 숫자로 시작해서는 안되고 특수문자는 ‘_’ , ‘$’만 허용됩니다.<br>(예: a, n2, max_num1→ 가능 3floor, #of→ 불가능)</p>
