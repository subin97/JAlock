<b>02. 데이터 형식1 : 정수와 실수</b></h4>

<p>문자열을 출력하는 방법을 배웠습니다! 다음은 자료형에 대해서 배우겠습니다.</p>
<p>먼저 정수와 실수를 표현할 수 있는 자료형을 소개해드리겠습니다.</p>
<p>1. int와 long은 정수를 나타내는데 차이점은 바로 범위입니다! int 데이터 형식은  -2,147,483,648 에서 2,147,483,647까지 나타내고 long은 –9,223,372,036,854,775,808에서 9,223,372,036,854,775,807까지 나타낼 수 있습니다. </p>
<p>2. float와 double은 실수를 나타내는데 유효 숫자의 개수에 따라 사용됩니다. float는 7개 정도의 유효 숫자를 갖고 double은 약 15개 정도의 유효 숫자를 가집니다. 예를 들어 582.021 은 유효숫자 6자리의 수입니다.</p>
