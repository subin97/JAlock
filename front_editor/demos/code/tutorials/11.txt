<b>04. if-else if-else문</b></h4>
<p>else if절을 이용하면 조건문의 흐름을 좀 더 자유롭게 제어할 수 있습니다.</p>
<p>if(조건문) { </p>
<p align="center"><수행할 문장1></p>
<p align="center">...</p>
<p>} else if(조건문) { </p>
<p align="center"><수행할 문장1></p>
<p align="center">...</p>
<p>} else if(조건문) { </p>
<p align="center"><수행할 문장1></p>
<p align="center">...</p>
<p>} else { </p>
<p align="center"><수행할 문장1></p>
<p align="center">...</p>
<p>} </p>
<p>if절의 값이 참이면 if문 바로 다음의 문장들이 실행되고 거짓이면 else if절로 제어가 넘어갑니다. else if절의 값이 참이면 else if문 바로 다음의 문장들이 실행되고 거짓이면 else절이 실행됩니다. else if절은 여러 개가 올 수 있고 else절은 생략이 가능합니다.
